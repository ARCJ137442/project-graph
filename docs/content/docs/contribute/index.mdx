### 安装 Rust 和 Node.js

- 安装 Rust：https://www.rust-lang.org/tools/install
- 安装 Node.js：https://nodejs.org/en/download/
- 安装 pnpm：`npm install -g pnpm`

### 安装项目依赖

```sh
pnpm install
```

### 启动应用

具体应该看 `package.json` 中的 `scripts` 字段。

应该能看出。这里之所以省略不写，是因为可能会变动，造成文档过时。

### 提交更改

推荐使用 Gitmoji 提交更改。（Vs Code插件）

- 安装 Gitmoji: `pnpm install -g gitmoji-cli`
- 提交更改: `gitmoji commit`

### 安装第三方库

由于本项目采用monorepo架构，涉及到增加完善功能，需要安装第三方库时，注意使用指令。

```sh
pnpm --filter @pg/app install decimal.js
```
