## 介绍

Project Graph 是一款专注于快速绘制节点图的桌面工具，旨在帮助用户高效地创建项目拓扑图和进行头脑风暴。用户可以通过简单的拖拽操作来创建和连接节点，从而轻松构建复杂的图结构。该工具还具备层级清晰的结构，能够直观地呈现项目关系或数据关系。

Project Graph 项目分为多个模块：

## 安装

### 下载安装包

<Callout>
如果你初次使用本应用，建议点击 `Release` 版本。

`Nightly` 版本是开发版本，带有最新的功能，但可能存在一些未知的 bug。
</Callout>

<GithubRelease />

### 从网盘下载

如果github或官网下载链接出了问题，可以尝试从网盘下载安装包。

https://pan.quark.cn/s/d4a90dba9491

由于网盘是手动备份，所以可能版本会落后于最新发布版本。

### AUR

如果你使用的是 Arch Linux，你可以通过 AUR 安装 Project Graph。

- [从 Nightly Release 安装] [project-graph-nightly-bin](https://aur.archlinux.org/packages/project-graph-nightly-bin)
- [从 Release 安装] [project-graph-bin](https://aur.archlinux.org/packages/project-graph-bin)
- [从源码安装，需要 Rust 环境] [project-graph-git](https://aur.archlinux.org/packages/project-graph-git)
