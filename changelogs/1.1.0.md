现在支持从命令行运行 Project Graph，并且打开一个工程文件：


## #156 自动保存和自动备份

此版本添加了备份功能，可在设置内调整自动保存时间间隔。

> [!IMPORTANT]
> 建议安装后第一时间设置一下草稿文件的备份路径。
>
> 两个计时的时间间隔计算方法是只有在窗口激活时才累计计时，窗口没有激活时不计时。
>
> 自动保存被触发时，舞台顶部会闪过一个白色光刃划过的特效。
>
> 在“最近文件”或“启动文件设置”面板处于打开状态时，自动保存会暂停
